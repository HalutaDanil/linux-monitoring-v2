# Linux Monitoring v2.0

> Продвинутый мониторинг с визуализацией в Grafana

## О проекте

Эволюция системы мониторинга: от простых bash-скриптов до полноценного дашборда в Grafana с нагрузочным тестированием.

## Что реализовано

| Задача | Описание |
|--------|----------|
| `01` | Валидация ввода: проверка числовых параметров |
| `02` | Генерация файлов заданного размера с заполнением |
| `03` | Очистка: удаление файлов по маске с подтверждением |
| `04` | Логирование: запись операций с timestamp |
| `05` | Сортировка: вывод top-N процессов по CPU/RAM |
| `06` | Визуализация в терминале (ASCII-графики) |
| `07` | Интеграция с **Grafana** + **Prometheus Node Exporter** |
| `08` | Нагрузочное тестирование: `stress` + мониторинг |
| `09` | Кастомный дашборд в Grafana |

## Стек мониторинга

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Сервер    │────►│  Prometheus │────►│   Grafana   │
│ Node Exporter│     │  (metrics)  │     │ (dashboard) │
└─────────────┘     └─────────────┘     └─────────────┘
```

## Grafana Dashboard

- CPU usage (user/system/iowait)
- RAM utilization
- Disk I/O
- Network throughput
- Load average

## Технологии

- **Bash** — автоматизация
- **Grafana** — визуализация метрик
- **Prometheus Node Exporter** — сбор системных метрик
- **stress** — нагрузочное тестирование

## Скриншоты

В `src/` сохранены скриншоты дашбордов Grafana под нагрузкой.
